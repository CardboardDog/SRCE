SRCE Engine Docs
explanation of .eyn files
-------------------------------
commands:
TITLE (Switch to title)
RLBUF (Render Level Buffer)
NAME (Set Level Name)
LWDTH (Level Width)
LHGHT (Level Height)
XSTRT (Player start x)
YSTRT (Player start y)
SCRED (Start Credits)
RBOOT (Reboot Game)
-------------------------------
buffers:
MBFER (Contains All Levels - Map Buffer)
LBFER (A Level Buffer)
LWALL (Level Wall Textures)
WHGHT (Wall Height)
END (Closes The Buffer)
SMODE (Game Settings)
TEXBF (Texture buffer)
GMTEX (Game Texture)
TEXST (Texture Set)
TEXPT (Texture Pointers)
